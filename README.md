# 🎸 기타 지판 트레이너

기타 지판의 수학적 모델을 활용한 인터랙티브 학습 도구입니다.

## 특징

- **수학적 정확성**: 12음계 모듈러 연산을 사용한 정확한 음 위치 계산
- **탐색 모드**: 특정 음의 모든 위치를 시각적으로 확인
- **퀴즈 모드**: 음 위치를 클릭하여 학습 효과 극대화
- **다양한 조율 지원**: Standard, Drop D, Open G, Open D 등
- **반응형 디자인**: 모바일과 데스크톱에서 모두 사용 가능

## 수학적 원리

### 핵심 공식

- **음높이 표현**: C=0, C#=1, ..., B=11 (12음계 모듈러)
- **프렛에서의 음**: `p ≡ sᵢ + f (mod 12)`
- **특정 음의 위치 찾기**: `f ≡ (n − sᵢ) (mod 12)`

### 표준 조율 (EADGBE)

- 6번줄 E → s₆=4
- 5번줄 A → s₅=9
- 4번줄 D → s₄=2
- 3번줄 G → s₃=7
- 2번줄 B → s₂=11
- 1번줄 E → s₁=4

## 사용법

### 탐색 모드

1. 목표 음을 선택합니다
2. 지판에서 해당 음의 모든 위치가 🎯로 표시됩니다
3. 프렛 범위를 조정하여 원하는 영역을 집중 학습할 수 있습니다

### 퀴즈 모드

1. "퀴즈" 버튼을 클릭하여 랜덤 음을 생성합니다
2. 지판에서 해당 음의 모든 위치를 클릭합니다
3. 정답은 초록색, 오답은 빨간색으로 표시됩니다
4. 모든 위치를 찾으면 완료됩니다
